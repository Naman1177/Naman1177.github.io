  • SafeSphere

    SafeSphere is an IoT-integrated smart helmet that provides 24/7 hazard monitoring for miners in high-risk underground environments. It uses LoRa technology for reliable "Deep Earth" communication and a sensor suite to detect toxic gases, low oxygen, and fire. The system includes Man-Down detection via an accelerometer to alert teams of falls and an IR sensor to ensure the helmet is actually being worn. By using frugal innovation, it delivers industrial-grade safety features at less than half the cost of traditional monitoring equipment.

    Github Repo - github.com/Naman1177/SafeSphere
